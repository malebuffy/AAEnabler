# AAEnabler

AAEnabler is an Android utility for selecting and installing Android Auto-compatible APK files stored on your device.

It is intended for apps that already support Android Auto but are distributed outside the usual app-store channels. AAEnabler does not modify APK files, add Android Auto support to ordinary apps, or include third-party applications.

## Features

- Select an APK from local device storage
- Inspect basic package and Android Auto compatibility information
- Warn when recognized Android Auto metadata is missing
- Open the Android installation flow
- Display a device-specific UUID
- Optional EUR 5.00 donation to support continued development

## Requirements

- Android 7.0 or newer
- An APK that is already compatible with Android Auto
- Permission to install apps from this source
- Internet access for donation and device-registration features

## Using AAEnabler

1. Open AAEnabler.
2. Tap **Select local APK** and choose a trusted Android Auto-compatible APK.
3. Review the detected application information and any compatibility warning.
4. Tap **Install app** and follow Android's installation prompts.
5. Reconnect or restart Android Auto if the newly installed app is not immediately visible.

## Donations

The optional EUR 5.00 donation supports AAEnabler's ongoing development. It does not purchase or include any third-party app.

During the donation flow, AAEnabler asks for an email address. After a successful donation, that address is associated with the device UUID for donation and licence identification. Payment details are handled by the payment provider and are not stored in the app.

## Important limitations

- AAEnabler cannot make an incompatible app work with Android Auto.
- APKs remain unchanged and keep their original signatures.
- Only install APKs obtained from sources you trust.
- Availability in Android Auto can still depend on the app, Android version, vehicle, and Android Auto configuration.
- AAEnabler is not affiliated with Google, Android Auto, or the developers of installed third-party apps.

## Disclaimer

Use AAEnabler responsibly and comply with applicable laws, platform rules, and vehicle-safety requirements. Never interact with applications while driving.
